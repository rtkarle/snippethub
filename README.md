# 📚 Online Code Snippet Manager
**Built for OnlineGDB Internship | Stack: NodeJS + MySQL + Bootstrap + jQuery**

A web application where users can save, manage, and share code snippets online — perfectly aligned with OnlineGDB's core product (online compiler/debugger).

---

## 🚀 Features
- User Registration & Login (JWT Authentication)
- Save, Edit, Delete code snippets
- Filter snippets by language (C, C++, Python, Java, etc.)
- Share snippet via public link
- Responsive UI with Bootstrap
- Dynamic search & filter using jQuery AJAX

---

## 🛠️ Tech Stack
| Layer | Technology |
|---|---|
| Backend | NodeJS + Express |
| Database | MySQL |
| Frontend | Bootstrap 5 + jQuery |
| Auth | JWT + bcrypt |
